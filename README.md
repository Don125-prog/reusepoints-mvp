# ReUsePoints MVP

ReUsePoints MVP — это прототип веб-платформы, объединяющей блокчейн-технологии и систему вознаграждений за экологичное поведение пользователей. Пользователи получают внутренние токены за передачу вещей для повторного использования и могут тратить их на бонусы от партнёров.


# Идея проекта

Цель проекта — мотивировать пользователей передавать вещи повторно, а не выбрасывать их.
За каждую передачу пользователь получает токены, которые затем можно обменять на бонусы внутри платформы.


# Основные возможности

* Регистрация и авторизация пользователей
* Автоматическая выдача blockchain-адреса
* Начисление токенов за передачу вещей
* Списание токенов при покупке бонусов
* Личный кабинет с балансом и историей операций
* Система достижений и ReUse Score
* AI-рекомендации для пользователя
* Интеграция с локальным блокчейном Ganache
* Смарт-контракт токена стандарта ERC-20

# Как работает система

## Регистрация

Пользователь регистрируется по email.
Система автоматически выдаёт свободный Ganache-кошелёк.

## Передача вещи

Пользователь указывает вещь для передачи.
Система сохраняет операцию и начисляет токены через смарт-контракт.

## Покупка бонуса

Пользователь выбирает бонус от партнёра.
Токены списываются с blockchain-баланса.

## Личный кабинет

Пользователь видит:

* баланс токенов
* историю операций
* достижения
* ReUse Score
* AI-рекомендации


# Установка и запуск

## 1. Клонировать репозиторий

```bash id="vgsxq9"
git clone https://github.com/your-username/reusepoints-mvp.git
cd reusepoints-mvp
```



## 2. Установить зависимости

```bash id="ttxgh0"
npm install
```

Установить Truffle глобально:

```bash id="fahlyq"
npm install -g truffle
```


## 3. Запустить Ganache

Открыть Ganache и запустить локальную сеть:

```text id="l74s9p"
127.0.0.1:7545
```


## 4. Скомпилировать смарт-контракт

```bash id="ab8qq5"
npx truffle compile --all
```


## 5. Задеплоить контракт

```bash id="1vd18l"
npx truffle migrate --reset --network development
```

Скопируйте адрес задеплоенного контракта.


## 6. Настроить `.env`

Создайте файл `.env`:

```env id="1a5ec7"
PORT=3000
GANACHE_HOST=127.0.0.1
GANACHE_PORT=7545
TOKEN_CONTRACT_ADDRESS=ВАШ_АДРЕС_КОНТРАКТА
SESSION_SECRET=your_secret_key
```


## 7. Запустить AI-сервис

```bash id="u2v8nr"
cd ai_service
python app.py
```


## 8. Запустить основной сервер

В новом терминале:

```bash id="zgn09p"
npm start
```

или

```bash id="0b7tzr"
node server.js
```


## 9. Открыть браузер

```text id="dt4fhv"
http://localhost:3000
```


# Роль смарт-контракта

Смарт-контракт отвечает за:

* выпуск токенов
* хранение балансов
* переводы токенов
* начисление наград
* списание токенов

---

# Роль blockchain.js

Файл `blockchain.js` обеспечивает связь сайта со смарт-контрактом и содержит функции:

* получение аккаунтов
* проверка баланса
* начисление токенов
* списание токенов
* получение данных токена



# Ограничения MVP-версии

* Используется локальная сеть Ganache
* JSON вместо полноценной базы данных
* Тестовая среда без реальных платежей
* Нет интеграции с MetaMask
